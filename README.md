# dotfiles-go

## Support Cipher Methods

- AES128_GCM
- AES256_GCM
- AES256_GCM_RAW
- AES128_CTR_HMAC_SHA256
- AES256_CTR_HMAC_SHA256
- CHACHA20_POLY1305
- XCHACHA20_POLY1305
